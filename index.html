CatalyXtraction

Idea

  • I wanted a way to extract the workouts from a workout routine on Catalyst Weightlifting website
  • I wanted to take these workouts and be able to view only a particular day's workout without having to scroll

Steps To Use

  • Download the HTML doc for almost any workout routing from Catalyst to a location you'll remember
  • Place the path to your HTML doc in the python script like below:
document = BeautifulSoup(open('path/to/your/workout/html'))

  • Run: python oly_program.py

  • Open extracted_workouts.html in Chrome

To Dos

  • Maybe write info to a csv file to view in Microsoft Excel too

Notes

This was a very minimalist project, so don't critique too much. I was just thinking of ways to use BeautifulSoup while learning a little python syntax. I use it myself and find it easier to use rather than scrolling through an entire workout routine.
